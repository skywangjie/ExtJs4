Notification extension for Ext JS 4.0.2+ (version: 2.1.3)

新改进的功能包括:

  • 多个栈管理器和通知
    静态管理器对象完全消除,让通知附加到不同的组件使用x和y坐标滑的通知。
  • 所有八个角落和边缘的文档/管理器可以使用:“tl”,“tr”,“br”,“bl”、”t“,”l“、”b“,”r“。
  • 可以使用x和y轴上的角落。通知可以在横向滑动。
  • 当一个通知任何通知后再次自动关闭。
  • 将鼠标停留在通知延迟自动关闭。
  • 容易定制的css、动画、持续时间、空隙/填充等。

安装指示:

检查这个演示页面的来源和自己做相应修改

  • 包括: Notification.js
     
  • 设定一个标准的填充和文本对齐通知页面上包含这个css: 
    .ux-notification-window .x-window-body {
	text-align: center;
	padding: 15px 5px 15px 5px;
}
  • 如果你想添加图标窗口标题你可以包括infoanderror图标和添加这个css样式表:
    .ux-notification-icon-information {
	background-image: url('images/icon16_info.png');
}
					
.ux-notification-icon-error {
	background-image: url('images/icon16_error.png');
}
    然后设置iconCls属性“ux-notification-icon-information”或“ux-notification-icon-error”启用通知上的图标。
     
  • 如果你想风格然后通知如下演示2中包括fader.png 并添加css样式表如下:
    .ux-notification-light .x-window-header {
	background-color: transparent;
}
					
body .ux-notification-light {
	background-image: url('images/fader.png');
}
					
.ux-notification-light .x-window-body {
	text-align: center;
	padding: 15px 5px 20px 5px;
	background-color: transparent;
	border: 0px solid white;
}
    然后设置cls属性“ux-notification-light”。 注意,这个样式有点新浏览器和不同 >=IE8.
